#### API Server与节点通信
![[截屏2025-11-28 17.48.02.png]]

| 交互场景           | 通信方式                         | 核心组件 / 协议           |
| -------------- | ---------------------------- | ------------------- |
| Pod / 容器生命周期管理 | kubelet → CRI → 容器运行时        | gRPC（Unix Socket）   |
| 容器健康检查         | kubelet → 容器网络（HTTP/TCP/ 命令） | HTTP/TCP/exec       |
| 日志采集           | kubelet 读取节点本地容器日志文件         | 文件系统（/var/log/pods） |
| 文件拷贝 / 端口转发    | kubelet 作为中介转发流量             | 本地端口转发 /exec        |
